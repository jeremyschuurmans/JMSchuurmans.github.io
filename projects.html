Companies I've Helped
  • GoLogic Solutions

    The Problem

    An exciting startup providing Chicago-area businesses with a pickup and delivery solution in the midst of the pandemic needed help realizing their broader vision for their React Native mobile app.

    The Solution

    As a junior developer on this project, my first task was to write tests for the endpoints in their Ruby on Rails API. Once the endpoints were fully tested, I implemented several front end and back end improvements, including adding PostGIS for a geolocation feature, and driving a substantial front end redesign. Among the improvements I delivered were a brand new landing page, filtered search results, and an improved shopping cart experience, helping to make the founder's vision for their app a reality, and support user growth outside of their Chicago-area base.

    Technologies Used

    Ruby on Rails, React Native, Heroku, Xcode, Android Studio

  • ThermoFisher Scientific

    The Problem

    A top laboratory equipment manufacturer needed help iterating and improving a successful internal sales tool.

    The Solution

    Here's something you don't see every day: a Ruby on Rails app with a MongoDB database. As a solo developer on this fascinating project, I integrated an existing serial number lookup tool into their app, and delivered initial work on a custom CMS, including full-stack development of an admin screen feature, and a customizable alert bar.

    Technologies Used

    JavaScript, Ruby on Rails, MongoDB, MongoDB Compass, Azure, Azure CLI

  • Handshake
